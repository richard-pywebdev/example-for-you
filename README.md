# example-for-you

## 01 Getting Started

This is an example project for build FastAPI web applications. 

### Usage

Using Fedora 38 Linux. 

### Install

```commandline
python3 -m venv venv
. venv/bin/activate
pip install --upgrade pip
pip install -r non_ver_reqs.txt
```

### Run

```commandline
python main.py
```
